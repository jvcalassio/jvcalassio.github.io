Fabian Liszt

Este é um projeto que cresceu aos poucos. Inicialmente, a tarefa era realizar mudanças em um tema, já existente, do WordPress. No entanto, o tema passou a não cumprir com as especificidades do projeto, e até atrapalhar por ter coisas demais que não seriam utilizadas.

Sendo assim, o novo rumo foi criar um tema novo, completamente do zero, com as funcionalidades necessárias para cumprir o projeto da forma desejada. Fiz o novo tema sem utilizar frameworks, apenas jQuery (que já vem incluso com o WordPress), e um plugin para acompanhar o tema e adicionar as funcionalidades de posts customizados.

A homepage do site foi um pequeno desafio, pois trabalha com diferentes vídeos em plano de fundo. Sendo assim, precisei otimizar ao máximo a forma com que os vídeos eram carregados e reproduzidos, para não deixar a página muito lenta no primeiro carregamento. Os videos carregam de forma inteligente, utilizando lazyloads para carregar apenas os necessários, sob demanda.

Para a página de projetos, foi implementada uma galeria utilizando o PhotoSwipe, além de previews ao deslizar/passar o mouse.

Já sobre o back-end do WordPress, foram criadas diversas funções de customização para ao cliente poder adaptar o site em diversos quesitos, como tamanhos e alinhamentos, caso queira.

O site também utiliza o Swup para fazer as transições de página de forma fluida.

Este projeto levou pouco mais de um mês, da ideia inicial até a conclusão.
